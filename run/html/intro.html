活动主题：“百舰奔月”——组织100艘战舰，齐发月球。

目标：地球到月球的距离384401公里，50人的航母，每人每年跑333公里就可以实现！“百舰奔月”活动，从北京时间2015年8月1日00:00开始，至2018年7月31日12:00结束。

 

意义：希望用跑步这种简单易行的方式，连接起全球系友的感情，唤起大家重视健康，虽然在不同的地点运动，但贵系人运动的心一直在一起，以此激励更多的人运动起来！！！三年之后的2018年，是贵系建系60周年，让我们用这种健康方式为母系献上一份特别的礼物。


记录：为了更快、更方便的在“奔月”活动中准确记录大家的运动成绩，系友杜凡学长的龙搜公司免费为该活动开发一个记录成绩的管理平台；使用方法如下：

 

1. 关注微信公众号“清华大学计算机系校友会”。

 

2. 进入公众号，点击“百舰奔月”之后，点击子菜单栏的上传成绩，出现注册页面，请填写个人信息以及需要加入的舰队，等待后台审批通过；通过之后即可上传成绩（选择锻炼方式、运动距离以及日期）；查看个人排行榜、舰队排行榜、总成绩等信息。

 

3. 后台统计里程=上传爬山里程乘2 /上传游泳里程乘4 /上传骑车里程除以3 /走路里程/跑步里程。

 

4. 快速进入公众号的方法：进入公众号，点击右上角的菜单栏中的添加到桌面。之后每次可以从手机桌面直接进入公众号。 ； 

 

5. 苹果手机用户上传里程，如果不能输入小数，请四舍五入。因为 输入文本框是可以输入小数的，但是有些手机的数字键盘没有”.”。


目标分解：为了更容易达成目标，同时能够调动大家运动热情。战舰发起有三种形式：


1. 由个人组织发起，名额50艘，每艘战舰至少要完成50000公里。 如：马少平老师要组织一艘战舰，就可以叫“识途号”，人员由马 老师组织。系友人员数量上限50人，参与人除贵系外，贵系家属 也欢迎加入，现有“家属陪伴号”。

 

2. 由班级、年级发起，名额30艘，每艘战舰至少要完成50000公里。 如：1995级，：“计95号”同样，系友人员数量上限50人，参与 人除贵系外，贵系家属也欢迎加入。

3. 由企业发起，名额20艘，每艘战舰至少要完成50000公里，同时   为该活动提供赞助。要求每艘战舰至少要有系友20人。如：“搜 狗号”。
